Project name: EcoTrack
Live Link: https://silver-madeleine-8f0d1b.netlify.app/

Features:
Users can join challenges such as CO₂ reduction, plastic avoidance, or green activities — all with personalized progress tracking.

Shows the user’s joined challenges categorized as Not Started, Ongoing, and Finished, with real-time statistics.

Login & Registration with Firebase Authentication, including Google Sign-In.

Beautifully designed interface using Tailwind CSS, optimized for mobile, tablet, and large screens.

All challenge data is fetched from a Node.js + Express + MongoDB server deployed on Vercel.
